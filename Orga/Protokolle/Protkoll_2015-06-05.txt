Grober Algorithmus für "create controlset":
{
	- 1 Bewegung als Tupel von Einstellungen (z.B. Geschwindigkeit, Achse, Strecke/Zeit)
	- Erstelle Map: Bewegungseinstlllung -> Koordinate (Entfernung ...)
	- Erstelle leeres Set
	- Für jeden Eintrag in der Map:
		Prüfe ob Koordinate+x schon in Set
			wenn nicht dann füge hinz
			sonst lösche Eintrag aus Map

	- Speichere Set als control-Set
}

Grober Algorithmus für "generate node-map"
{
	- markiere aktuelle pos als ersten node
	
	- mache folgendes:
		- für jede bewegung in control-set
		führe bewegung aus
		ist akutelle pos schon in node liste?
			wenn nicht, hänge pos als node an
		führe "gegenbewegung aus"

		fahre zum nächsten node

	- solange wie aktuelle pos nicht letzte pos auf liste
}
